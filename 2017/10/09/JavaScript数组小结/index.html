1. 栈方法 （后进先出）

  • pop() 从数组的末尾移除最后一项，减少数组的length值,返回值为被移除的项。

操作方法

  • concat() 不影响原数组，基于当前数组复制一个出来，不传参的情况下，返回原数组的副本。 传参的情况下，可以传一个或多个数组，然后这些数组项会被挨个添加到结果数组，并返回。

  • slice() 不会改变原数组， 可以接受一个或两个参数。如果有只有一个参数，则该方法返回从该参数指定位置到当前数组的末尾所有项。两个参数的情况下，该方法返回起始位置和结束位置之间的项但并不包括结束位置的项。

注意：如果slice() 的参数有一个负数，则数组长度加上该数来确定想应的位置，例如，假设数组长度为6，slice(-2, -1) 与调用slice(4,5),得到的结果相同。如果结束位置小于开始位置，则返回空数组。

  • splice() 主要向数组中插入项，会改变原数组。
    • 删除：删除任意数量的项，只需要指定2各参数：要删除的第一项的位置和要删除的项目。
    • 插入：可以向指定位置插入任意数量的项，3个参数：起始位置、0（要删除的项目）、要插入的项。
    • 替换：可以向指定位置插入任意数量的项，且同时删除任意数量的项。指定3个参数：起始位置、要删除的项目、要插入的任意数量的项。

位置方法

  • indexOf()和lastIndexOf() 可接受两个参数： 要查找的项、查找起点位置的索引（可选项目）。前者从开始向后查找，后者从数组的末尾向前开始查找。

这两个方法返回查找的项在数组中的位置，没找到的情况下返回 -1

迭代方法

every()、filter()、foreEach()、map()、some()、每个方法可以接受两个参数：1.每一项运行的函数 2.（可选的）运行该函数的作用域对象。该函数接受3个参数（数组项的值、该项在数组中的位置、数组对象本身）。

  • every() ：对数组每一项运行指定函数，该函数的每一项都返回true，则返回true。
  • filter()：对数组每一项运行指定函数，会返回一个由该函数返回true的结果组成的一个数组。
  • forEach()：对数组每一项运行指定函数，这个方法没有返回值。
  • map()：对数组每一项运行指定函数，返回每次函数调用结果组成的数组。
  • some()：对数组每一项运行指定函数，如果该函数对任一项返回true,则返回true。

归并方法

  • reduce() 和reduceRight(),方法可以接受一个函数并且接受4个参数：前一个值、当前值、项的索引、数组对象。这个函数返回的任何值都会作为第一个参数自动传递给下一项，第一次迭代发生在数组的第二项上，因此第一个参数是数组的第一项，第二个参数就是数组的第二项。

Array对象常用方法中：·

不改变原数组：

1、 concat()

  • 连接两个或多个数组
  • 不改变原数组
  • 返回被连接数组的一个副本

2、join()

  • 把数组中所有元素放入一个字符串
  • 不改变原数组
  • 返回字符串

3、 slice()

  • 从已有的数组中返回选定的元素
  • 不改变原数组
  • 返回一个新数组

4、 toString()

  • 把数组转为字符串
  • 不改变原数组
  • 返回数组的字符串形式

改变原数组：

5、 pop()

  • 删除数组最后一个元素，如果数组为空，则不改变数组，返回undefined
  • 改变原数组
  • 返回被删除的元素

6、 push()

  • 向数组末尾添加一个或多个元素
  • 改变原数组
  • 返回新数组的长度

7、 reverse()

  • 颠倒数组中元素的顺序
  • 改变原数组
  • 返回该数组

8、 shift()

  • 把数组的第一个元素删除，若空数组，不进行任何操作，返回undefined
  • 改变原数组
  • 返回第一个元素的值

9、 sort()

  • 对数组元素进行排序(ascii)
  • 改变原数组
  • 返回该数组

10、 splice()

  • 从数组中添加/删除项目
  • 改变原数组
  • 返回被删除的元素

11、 unshift()

  • 向数组的开头添加一个或多个元素
  • 改变原数组
  • 返回新数组的长度
